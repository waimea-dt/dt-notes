# Testing Material for MkDocs

Trying out [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/)

Docs are [here](https://squidfunk.github.io/mkdocs-material/reference/)

## Hosted

Hosted at [https://waimea-cpy.github.io/mkdocs-material-test](https://waimea-cpy.github.io/mkdocs-material-test), built via GitHub Actions using [this config](https://squidfunk.github.io/mkdocs-material/publishing-your-site/#with-github-actions)

## Local Dev.

Install locally via:
```
pip install -r requirements.txt
```

Then serve from the site folder with:
```
mkdocs serve
```

Build the static site:
```
mkdocs build
```
